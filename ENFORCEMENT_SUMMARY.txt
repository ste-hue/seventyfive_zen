THE 4 GATES
─────────────────────────────────────────────────────────────────

GATE 1: State Coherence Check
  → Blocks action if inner state < 5
  → Forces state reset before proceeding
  → "Action from noise compounds noise"

GATE 2: Causality Chain
  → Intention → Attention → Action → Result
  → Each layer must trace to previous
  → Rejects vague/unanchored language

GATE 3: Backward Debug
  → Triggered when edge < 5
  → Result → Action → Words → Attention → State
  → Identifies root cause mechanically

GATE 4: Daily Coherence Check
  → Once per day: "State consistent with actions?"
  → 10 second check
  → Prevents drift
